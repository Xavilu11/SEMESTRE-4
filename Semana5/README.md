# Galería Interactiva con JavaScript

Este proyecto implementa una galería dinámica de imágenes usando **HTML, CSS y JavaScript**.

## Funcionalidades
- Agregar imágenes mediante URL.
- Seleccionar una imagen (solo una a la vez).
- Eliminar la imagen seleccionada.
- Atajo de teclado: Enter para agregar imágenes.
